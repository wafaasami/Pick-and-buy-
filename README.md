# Pick and Buy – Home Appliances Marketing Plan 🏠🛒


## 🔗 Project Description

This project focuses on creating and managing a digital marketing strategy for **"Pick and Buy"**, a social media brand specialized in modern home appliances and kitchenware. The team is responsible for building a strong brand identity, showcasing high-quality home tools, and reaching homeowners through effective digital marketing channels to simplify their daily lives.

All team members collaboratively worked on:
